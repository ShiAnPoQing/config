跳转到标记 

    1. ` 光标被定位到指定位置
               并且该移动是 |exclusive|（排他性）的。
    2. ' 光标被定位到指定位置所在行的第一个非空白字符
               并且该移动是行方式的。
    3. g` 或 g'

1. {a-z} 小写标记 
2. {A-Z} 跨文件标记
3. {0-9} 编号标记(shada，不能直接设置)
4. "'" 或 "`" 上下文标记
5. [ 和 ] 上一次更改或复制文本的起止位置 
6. < 和 > 上一次选择的起止位置(`gv`)
